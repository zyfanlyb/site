# 关于

这是一个基于 VitePress 构建的网站前端项目。

## 技术栈

- **VitePress**: 静态站点生成器
- **Vue 3**: 渐进式 JavaScript 框架
- **Vite**: 下一代前端构建工具

## 项目结构

```
front_service/
├── .vitepress/          # VitePress 配置
│   └── config.js       # 配置文件
├── guide/              # 指南文档
├── about/              # 关于页面
└── index.md            # 首页
```
